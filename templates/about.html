About MorseTyper

What is Morse Code? Morse code is a method of telecommunication in which text characters are expressed through the combination of 2 different signals: "dots" and "dashes". The duration of a dot is 1 unit, while the duration of a dash is 3 units. The space betwen parts of the same letter is one unit; the space between different letters is 3 units; the space between different words is 7 units.

History of Morse Code: The morse code system is named after Samuel Morse, who invented the telegraph during the 1930s. In a time when technology was not advanced enough to transmit human voices, the telegraph allowed for long-distance communication through the use of morse code. Morse code was crucial during times like WWII, the Korean and Vietnam Wars, and more. Morse code required operators who could quickly express and decipher phrases in morse code, which required an immense amount of practice.

How to use: MorseTyper is intended to teach the user to communicate morse code in a controlled environment, where they can track their speed and proficiency.

  • When the game begins, a sentence will appear on the screen, which the player must express using morse code.
  • Press spacebar to input the morse code. The current letter will be highlighted in the sentence.
  • The time will be kept track of in the bottom left of the screen.
  • The player can choose to see the morse code chart by clicking the checkbox.
